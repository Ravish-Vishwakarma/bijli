# ⚡ BIJLI

> **Your Memes. At Light Speed.**

BIJLI is a lightning fast, keyboard centric launcher designed to put your entire local meme collection just one shortcut away. Stop digging through nested folders hit the hotkey, type, and copy.

## ✨ Features

- **🚀 Instant Access**: Toggle the launcher instantly from anywhere with `Ctrl+Shift+M`.
- **📂 100% Local**: Works with your existing local folders. No cloud uploads, no privacy concerns.
- **🔍 Fuzzy Search**: Find the perfect reaction GIF instantly, even if you make a typo.
- **⌨️ Keyboard First**: Designed for speed. Open, navigate (`↑` `↓` `←` `→`), and copy (`Enter`) without touching your mouse.
- **📋 One-Click Copy**: Images are copied directly to your clipboard, ready to paste into Discord, Slack, or X.
- **⚡ Zero Latency**: Optimized for performance to handle large collections effortlessly.
## 🛠️ Tech Stack

- **Frontend**: React, Tailwind CSS, Framer Motion
- **Core**: Rust (Tauri) for native performance and system integration
- **Search**: Fuse.js for client-side fuzzy matching

## 🚀 Usage

1. **Select Folder**: Point BIJLI to your local memes directory.
2. **Launch**: Press `Ctrl+Shift+M`.
3. **Search**: Start typing to filter results.
4. **Copy**: Hit `Enter` to copy the selected meme.
